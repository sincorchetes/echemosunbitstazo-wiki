Spotify
-------

Introducción
############

Software privativo que permite escuchar música en streaming, si posees cuenta Premium puedes incluso mantener la música descargada en tu dispositivo.

.. note::

  Requisitos: 
  * Tener instalado los repositorios RPM Fusion

Instalación
###########

.. code-block:: bash

  sudo dnf install lpf-spotify-client
  lpf-gui

Ejecutar
########

.. code-block:: bash

  spotify
