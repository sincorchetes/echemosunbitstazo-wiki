Zoom
----

Introducción
############

Software privativo que permite realizar videollamadas, llamadas telefónicas a través de Vo-IP entre otras cosas.

Instalación
###########

.. code-block:: bash

  curl -LO https://zoom.us/client/latest/zoom_x86_64.rpm
  sudo dnf install zoom_x86_64.rpm

Ejecutar
########

.. code-block:: bash

  zoom
