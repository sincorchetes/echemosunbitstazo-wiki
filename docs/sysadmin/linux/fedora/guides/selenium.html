Selenium¶

Google Chrome y Python¶

  • Requisitos: * Tener instalado Google Chrome * Tener instalado el plugin de Selenium IDE * Realizar una grabación o acciones en una página * Exportar el .side a .py desde el plugin de Chrome

sudo dnf install chromedriver

Crear Virtualenv

python -m venv selenium

Instalar dependencias

source selenium/bin/active
pip install selenium pytest

Editar el archivo exportado *.py y añadir lo siguiente:

o = CLASE()
o.setup_method('get')
o.METODO()

Hay que cambiar CLASE y METODO según lo que hay en el archivo y ejecutar el script

python NombreArchivo.py
