Skype
-----

Introducción
############

Software privativo que permite realizar videollamadas, llamadas telefónicas a través de Vo-IP entre otras cosas.

Instalación
###########

.. code-block:: bash

  curl -LO https://go.skype.com/skypeforlinux-64.rpm
  sudo dnf install skypeforlinux-64.rpm

Ejecutar
########
.. code-block:: bash

  skypeforlinux
