MariaDB - Comandos SQL
----

Bases de datos
####

Crear base de datos
****

.. code-block:: sql
  CREATE DATABASE nombre_base_de_datos;


Eliminar base de datos
****

.. code-block:: sql
  DROP DATABASE nombre_base_de_datos;


Mostrar las bases de datos
****

.. code-block:: sql
  SHOW DATABASES;


Situarse en la base de datos
****

.. code-block:: sql
  USE nombre_base_de_datos;
